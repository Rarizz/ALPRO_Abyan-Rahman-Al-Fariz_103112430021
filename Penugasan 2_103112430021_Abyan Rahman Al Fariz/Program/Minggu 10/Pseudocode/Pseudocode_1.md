Program Cek Jenis Segitiga

Kamus
    a, b, c : integer

Algoritma
    Input a
    Input b
    Input c

    If a == b and b == c
        Output "Segitiga sama sisi"
    Else If a == b or b == c or a == c
        Output "Segitiga sama kaki"
    Else
        Output "Segitiga sembarang"
    End If
End Program
