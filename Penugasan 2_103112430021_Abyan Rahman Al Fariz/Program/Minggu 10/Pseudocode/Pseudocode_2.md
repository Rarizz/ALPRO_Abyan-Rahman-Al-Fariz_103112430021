Program Cek Stabilitas Nilai

Kamus
    t1, t2, t3, t4, t5 : real

Algoritma
    Input t1
    Input t2
    Input t3
    Input t4
    Input t5

    If t1 < t2 and t2 < t3 and t3 < t4 and t4 < t5
        Output "Stabil naik"
    Else If t1 > t2 and t2 > t3 and t3 > t4 and t4 > t5
        Output "Stabil turun"
    Else
        Output "Tidak stabil"
    End If
End Program
