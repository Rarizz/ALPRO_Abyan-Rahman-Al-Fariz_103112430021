Program Cetak Persegi Angka

Kamus
    angka, i, j : integer

Algoritma
    Input angka

    For i = 1 to angka
        Output newline

        For j = 1 to angka
            If i == 1 or i == angka or j == 1 or j == angka
                Output i with space
            Else
                Output two spaces
            End If
        End For
    End For
End Program
